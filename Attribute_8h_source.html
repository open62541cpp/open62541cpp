26/**
27 * @defgroup Attribute Attribute service set
28 * Read and write node attributes.
29 *
30 * The following node attributes cannot be changed once a node has been created:
31 * - NodeClass
32 * - NodeId
33 * - Symmetric
34 * - ContainsNoLoops
35 *
36 * The following attributes cannot be written from the server, as they are specific to the different
37 * users and set by the access control callback:
38 * - UserWriteMask
39 * - UserAccessLevel
40 * - UserExecutable
41 *
42 * @see https://www.open62541.org/doc/1.3/server.html
43 * @see https://reference.opcfoundation.org/Core/Part4/v105/docs/5.10
44 * @ingroup Services
