26/**
27 * High-level node class to access node attribute, browse and populate address space.
28 *
29 * The Node API is just a more convenient way of using the free functions in the `services`
30 * namespace.
31 *
32 * Node objects are useful as-is but they do not expose the entire OPC UA protocol. You can get
33 * access to the associated NodeId instance with the getNodeId() method and apply the native
34 * open62541 functions or the free functions in the `services` namespace.
35 *
36 * @see Services
37 */
