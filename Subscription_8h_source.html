42/**
43 * High-level subscription class.
44 *
45 * The API is symmetric for both Server and Client, although servers don't use the subscription
46 * mechanism of OPC UA to transport notifications of data changes and events. Instead MonitoredItems
47 * are registered locally. Notifications are then forwarded to user-defined callbacks instead of a
48 * remote client. The `subscriptionId` for servers is always `0U`.
49 *
50 * @note Not all methods are available and implemented for servers.
51 *
52 * Use the free functions in the `services` namespace for more advanced usage:
53 * - @ref Subscription
54 * - @ref MonitoredItem
55 */

57class Subscription {
