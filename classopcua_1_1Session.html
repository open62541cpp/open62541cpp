opcua::Session Class Reference

#include <Session.h>

Detailed Description

High-level session class to manage client sessions.

Sessions are identified by a server-assigned sessionId (of type NodeId). A session carries attributes in a key-value list. Custom attributes/meta-data can be attached to a session as key-value pairs of QualifiedName and Variant.

See also
https://reference.opcfoundation.org/Core/Part4/v105/docs/5.6
