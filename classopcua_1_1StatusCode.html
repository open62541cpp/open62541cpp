Detailed Description

UA_StatusCode wrapper class.

This type is not derived from TypeWrapper. UA_StatusCode is just an alias for uint32_t. StatusCode can be used interchangeably with UA_StatusCode due to implicit conversions (without any overhead) but provides some methods to simplify the handling with status codes.
