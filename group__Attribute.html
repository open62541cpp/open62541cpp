Detailed Description

Read and write node attributes.

The following node attributes cannot be changed once a node has been created:

  • NodeClass
  • NodeId
  • Symmetric
  • ContainsNoLoops

The following attributes cannot be written from the server, as they are specific to the different users and set by the access control callback:

  • UserWriteMask
  • UserAccessLevel
  • UserExecutable
See also
https://www.open62541.org/doc/1.3/server.html
https://reference.opcfoundation.org/Core/Part4/v105/docs/5.10
