Detailed Description

template<typename ServerOrClient>
class opcua::Node< ServerOrClient >

High-level node class to access node attribute, browse and populate address space.

The Node API is just a more convenient way of using the free functions in the services namespace.

Node objects are useful as-is but they do not expose the entire OPC UA protocol. You can get access to the associated NodeId instance with the getNodeId() method and apply the native open62541 functions or the free functions in the services namespace.

See also
Services
Examples
client_browse.cpp, client_minimal.cpp, and server_minimal.cpp.

Write the MinimumSamplingInterval attribute of a variable node.

Specifies (in milliseconds) how fast the server can reasonably sample the value for changes. A minimum sampling interval of 0 indicates that the server is to monitor the item continuously. A minimum sampling interval of -1 means indeterminate.
