17 * Template base class to wrap `UA_*` type objects.
18 *
19 * Zero cost abstraction to wrap the C API objects and delete them on destruction. The derived
20 * classes should implement specific constructors to convert from other data types.
21 *
22 * @warning No virtual constructor defined, don't implement a destructor in the derived classes.
23 * @ingroup Wrapper
24 */
